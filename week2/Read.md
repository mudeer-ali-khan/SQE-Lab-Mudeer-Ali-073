# ISO 25010 Quality Attribute Analysis: LinkedIn vs. Indeed

## 📌 Project Overview
This project is part of the **Software Quality Engineering (SQE)** Lab at Mirpur University of Science & Technology (MUST). The goal is to perform a comparative quality assessment of two mobile applications—**LinkedIn** and **Indeed**—based on the **ISO/IEC 25010** software quality model.

The analysis focuses on eight key quality characteristics to identify trade-offs between feature richness and performance efficiency.

## 📊 Quality Characteristics Evaluated
1. **Functional Suitability**
2. **Performance Efficiency**
3. **Compatibility**
4. **Usability**
5. **Reliability**
6. **Security**
7. **Maintainability**
8. **Portability**

## 🛠️ Tech Stack & Tools
- **Language:** Python 3.x
- **Libraries:** `matplotlib`, `numpy` (used for data visualization)
- **Environment:** VS Code / Jupyter Notebook
- **Model:** ISO/IEC 25010 Quality Model
- **Testing Tools:** iOS Device (iPhone 11), Stopwatch, Network Analytics

## 📈 Visualizing the Results
The project includes a Python script that generates a **Radar Chart** (Spider Chart) to visualize the quality profiles of both apps simultaneously. This allows for a quick visual comparison of where one app outperforms the other.
